# OJ Version

This folder contains the console / OJ submission version of the snake assignment.

## Files

- `main.c`: recommended C submission version
- `main.cpp`: C++ version kept for reference

## Notes

- The OJ version follows the interactive I/O requirement from the assignment.
- It prints one move direction and the score before each step.
- It calls `fflush(stdout)` after each output round.
- When the judge returns `100 100`, it outputs the map state before collision and the final score.

## Suggested submission

If your course platform accepts C, submit `main.c`.
